Available Commands

Create Vault

Create a new encrypted Ethereum key vault with a strong master password.

eth-cli create-vault --name [VAULT_NAME]

Configure Cloud Storage

Connect your vault to cloud storage providers for secure backups.

eth-cli config --cloud [PROVIDER] --auth

Backup Vault

Manually trigger a vault backup to configured cloud storage.

eth-cli backup --vault [VAULT_NAME]

Restore Vault

Restore your encrypted vault from cloud storage.

eth-cli restore --from [PROVIDER] --vault [VAULT_NAME]

Verify Vault Integrity

Check the integrity and security of your vault.

eth-cli verify --vault [VAULT_NAME]
